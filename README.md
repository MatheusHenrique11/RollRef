# 🛼 RollRef — Referência em Patins

Portal definitivo de referência, comparação e comunidade 
para o universo dos patins no Brasil.

## Sobre
RollRef conecta patinadores a informações técnicas confiáveis,
comparações detalhadas de modelos e avaliações reais da comunidade.
Lojistas e montadores podem divulgar seus produtos para um público
altamente qualificado.

## Funcionalidades
- Catálogo com filtros por categoria, marca e nível
- Comparador de até 3 modelos lado a lado
- Sistema de avaliações da comunidade
- Portal de lojistas com planos de publicação

## Stack
Next.js 14 · JavaScript · CSS Vanilla · PostgreSQL (Supabase)
